## Core User Flows

Launch app (splash)
↓
Choose date and destination (home)
↓
Transition screen (travel-transition)
↓
View headlines (newspaper)
↓
Read article (newspaper-article)

From newspaper-article, user can go back to newspaper.
From newspaper, user can go back to home.
